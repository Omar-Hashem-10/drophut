# Drophut

**Drophut** is a project developed by **EraaSoft**. It is an all-in-one e-commerce platform designed to provide users with a seamless shopping experience, along with an admin dashboard for managing products, orders, feedback, and more.

## Website Content

### 1. **Home Page**
The home page includes several sections:
- **Tranding Products**: Displaying the most popular products.
- **Awesome Features**: Showcasing the special features of the website.
- **Video Product**: Featuring videos of products.
- **Other Collections**: Showcasing various product collections.
- **Feedback**: Customer feedback and reviews.
- **Blog Posts**: Displaying the latest blog articles.

### 2. **Other Pages**:
- **Products**: A page for browsing products.
- **About**: Information about the store.
- **Blogs**: Blog section with various articles.
- **Contact**: A contact page for users to get in touch with the site.
- **Profile**: User profile page.
- **Wishlist**: A list of favorite products.
- **Cart**: Shopping cart.
- **Order**: View current and past orders.

## Dashboard

The site includes a comprehensive dashboard for admins, employees, and customers with different access levels. The dashboard contains the following sections:

### 1. **Users**:
- **Customers**: View and manage customer accounts.
- **Admin**: View and manage admin accounts.
- **Employee**: View and manage employee accounts.

### 2. **Products**:
- **View Products**: Display all products.
- **Add Product**: Add new products to the inventory.

### 3. **Blogs**:
- **View Blogs**: Display all blog posts.
- **Add Blog**: Add new blog articles.

### 4. **Orders**:
- **View Orders**: View and manage customer orders.

### 5. **Categories**:
- **View Categories**: Display all product categories.
- **Add Category**: Add new categories.

### 6. **Reviews and Comments**:
- **Review for Products**: Manage product reviews.
- **Comments for Blogs**: Manage comments on blog posts.

### 7. **Feedback**:
- **View Feedback**: View customer feedback and reviews.

### 8. **Social Media**:
- **View Social Media**: Manage social media links.
- **Add Social Media**: Add new social media profiles.

### 9. **Description**:
- **View Description**: Display existing descriptions.
- **Add Description**: Add new descriptions for products or sections.

---

## Technologies Used

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: PHP (with Object-Oriented Programming - OOP)
- **Database**: MySQL
- **Version Control**: Git

The project is built using **Object-Oriented Programming (OOP)** principles to organize code into classes and objects, making the application more modular, scalable, and maintainable.
